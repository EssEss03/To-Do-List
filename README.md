# 📌 To-Do List Application

A simple, elegant, and fully functional To-Do List application built with **HTML, CSS, and JavaScript**. This app allows users to **add, edit, complete, and delete tasks** while organizing them by **categories and due dates**. Tasks persist even after a page refresh using **localStorage**.

---

## 🚀 How to Run the Project

### 🔹 Option 1: Open Locally in a Browser
1. Clone or download the repository to your local machine:
   ```bash
   git clone <your-repo-url>

2. Navigate into the project folder:
   bash
   cd to-do-list

3.Open index.html in a web browser (Chrome, Firefox, Edge, etc.)
   You can double-click the index.html file or drag and drop it into your browser.
   
Option 2: Run with a Local Server
For better development and testing, use live-server (Node.js required).
 1. Install live-server (if not installed):
   bash
   npm install -g live-server

 2. Run the project:
   bash
   live-server

3. The application will open automatically in your browser.

📝 How to Use the To-Do List 📝
1️⃣ Enter a task in the input field.
2️⃣ Pick a due date from the date selector.
3️⃣ Choose a category (Work, Personal, Urgent, or Other).
4️⃣ Click "Add Task" – Your task will be added to the list.
5️⃣ Mark a task as completed by clicking ✔ – The task moves to the "Completed Tasks" section.
6️⃣ Edit a task by clicking ✎ – Modify the task name, due date, or category.
7️⃣ Save changes by clicking 💾 – Updates will be saved immediately.
8️⃣ Delete a task by clicking ✖ – The task will slide out and be permanently removed.
9️⃣ Tasks persist even after you refresh the page!

✨ Features Included ✨
✅ Add, Complete, Edit, and Delete Tasks – Manage tasks easily.
✅ Task Categories – Organize tasks under Work, Personal, Urgent, or Other.
✅ Due Dates – Set and track deadlines.
✅ Completed Tasks Section – Automatically moves completed tasks to a separate section.
✅ Persistent Storage – Uses localStorage to save tasks even after closing the browser.
✅ Animations & UI Effects – Smooth, modern, and responsive UI with animations.
✅ Mahogany Red & Syrup Brown Theme – Aesthetically pleasing color scheme.

🎬 Animations & Effects 🎬
🟣 Fade-in Animation for Adding Tasks – When a task is added, it smoothly fades in.
🟣 Slide-out Animation for Deleting Tasks – When deleting a task, it slides out before disappearing.
🟣 Bounce Effect on Task Completion – Clicking the ✔ button makes the task slightly bounce.
🟣 Hover Effects for Buttons – Buttons smoothly change size & color when hovered.
🟣 Shake Effect for Overdue Tasks – Overdue tasks shake slightly to grab attention.
